### Vocabulary Table Structure Rules

When creating vocabulary tables, follow this structure:

1. **Column Order** (left to right):

2. **Table Header Format**:

## Key Language Comparisons (Greek ‚Üî Italian)

### Similarities

- Both languages have grammatical gender (masculine/feminine; Italian also has neuter forms in some contexts)
- Verb conjugations based on person and number
- Similar Latin-derived vocabulary in many domains
- Use of articles before nouns

### Key Differences

- Italian has 2 genders (m/f), Greek has 3 (m/f/n)
- Italian uses Latin alphabet, Greek uses Greek alphabet
- Different pronunciation systems
- Italian has simpler case system (Greek has 4 cases)
